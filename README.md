# Late e Mia - Frontend
Frontend utilizado para gerenciar agendamentos de consultas veterinárias.

## Instalar
- Na pasta do projeto, rodar comando `npm install`

## Variáveis de ambiente
Na raiz do projeto, criar um arquivo chamado `.env` com as seguintes variáveis:
```
NEXT_PUBLIC_API_URL= http://localhost:3000 # URL da API
```

## Rodar
- Na pasta do projeto, rodar `npm run dev`
